Features:

How to add this to your website:

Simply add this one line of code before the closing </body> tag:

<script src="http://localhost:5000/chatbot-embed.js"></script>

Customization:

You can customize the widget by modifying the configuration in the script:

// Change colors, position, title, etc. const config = { primaryColor: '#f3202e', // Your brand color position: 'bottom-right', // or 'bottom-left' title: 'AI Assistant' // Widget title };
